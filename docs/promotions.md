# Акции

## Описание

Акция — временное предложение, которое применяется ко всей корзине в виде процента скидки от суммы.

В отличие от скидки на корзину (порог по сумме), акция действует по времени.

## Сущность "Акция"

| Поле            | Тип      | Описание                                           |
| --------------- | -------- | -------------------------------------------------- |
| id              | UUID     | Уникальный идентификатор                           |
| name            | string   | Название (Новогодняя акция, Чёрная пятница и т.д.) |
| description     | string   | Описание акции                                     |
| percent         | int      | Процент скидки (%)                                 |
| date_from       | datetime | Дата начала действия                               |
| date_to         | datetime | Дата окончания                                     |
| is_active       | boolean  | Активна и доступна клиентам                        |
| created_user_id | UUID     | ID пользователя, создавшего акцию                  |
| created_at      | datetime | Дата создания                                      |
| updated_at      | datetime | Дата обновления                                    |

## Пример справочника

| Название       | description   | percent | date_from        | date_to          | is_active |
| -------------- | ------------- | ------- | ---------------- | ---------------- | --------- |
| Новогодняя     | Праздничная   | 10%     | 2024-12-25 00:00 | 2025-01-05 23:59 | true      |
| Чёрная пятница | Успей купить  | 15%     | 2024-11-29 00:00 | 2024-11-30 23:59 | false     |
| Весенняя       | Скидки весной | 5%      | 2025-03-01 00:00 | 2025-03-31 23:59 | false     |

## Валидация

| Правило             | Описание                    |
| ------------------- | --------------------------- |
| percent 1-100       | Процент скидки от 1 до 100  |
| date_to > date_from | Дата окончания позже начала |
| Уникальность name   | Название акции уникально    |
