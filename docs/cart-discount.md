# Скидки на корзину

## Описание

Скидка от суммы заказа — применяется ко всей корзине при достижении порога суммы.

> **Важно:** Это **НЕ** тип цены товара, а отдельная сущность. Не отображается в таблице цен при создании товара.

## Сущность "Скидка на корзину"

| Поле      | Тип     | Описание                                 |
| --------- | ------- | ---------------------------------------- |
| id        | UUID    | Уникальный идентификатор                 |
| name      | string  | Название скидки                          |
| min_sum   | decimal | Минимальная сумма для применения         |
| percent   | int     | Процент скидки (%)                       |
| apply_to  | enum    | all / retail / wholesale — кому доступно |
| is_active | boolean | Активна и доступна клиентам              |

## apply_to (кому доступно)

| Значение  | Описание                   |
| --------- | -------------------------- |
| all       | Всем (розничные + оптовые) |
| retail    | Только розничным           |
| wholesale | Только оптовым             |

## Пример справочника

| Название      | min_sum   | percent | apply_to  |
| ------------- | --------- | ------- | --------- |
| Скидка от 10к | 10 000 ₽  | 10%     | retail    |
| Скидка от 20к | 20 000 ₽  | 15%     | retail    |
| Скидка от 50к | 50 000 ₽  | 20%     | all       |
| VIP клиент    | 100 000 ₽ | 25%     | wholesale |
