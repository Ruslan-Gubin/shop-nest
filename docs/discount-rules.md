# Правила применения скидки для общей корзины

## Общие правила

- Всегда применяется **ОДНА лучшая скидка** (max%)
- Скидки **НЕ суммируются**
- Скидка на корзину и Акция конкурируют между собой

## Сравнение: Акция vs Скидка на корзину

| Критерий          | Акция                     | Скидка на корзину      |
| ----------------- | ------------------------- | ---------------------- |
| Условие           | Время (validFrom/validTo) | Сумма корзины (minSum) |
| Когда применяется | В даты акции              | При достижении порога  |
| Тип скидки        | % от суммы                | % от суммы             |

## Алгоритм применения

1. Рассчитываем сумму корзины по РОЗНИЧНЫМ ценам:
   baseCartSum = sum(товар × розничная цена)

2. Находим активные акции (по текущему времени) акция может быть только одна:
   promotion = promotions.percent ?? 0

3. Находим скидки на корзину (по сумме):
   cartDiscount = от baseCarSum ищем минимальную суммы скидки на корзину и подходящий по типу пользователя (розница или пользователяк) также isActive = true

4. Выбираем одну лучшую скидку:
   bestDiscount = MAX(promotion, cartDiscount)

5. Применяем:
   finalSum = baseCartSum × (1 - bestDiscount/100)
